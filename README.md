# Personal website of Jeremias Erbs

## Used technology
- parcel
- posthtml
- postcss
- babel
- nanocss


## Installation

```shell
$ npm install
```

## Develop and run server
```shell
$ npm start
```

## Deploy for Github Pages or local view
```shell
$ npm run build
```
This will create a new version of the `docs/` folder. Push to `master`

## Anything else
Look into the package.json
